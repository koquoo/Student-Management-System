Student Management System (CRUD)

A simple web-based Student Management System built using PHP, MySQL (XAMPP), HTML, CSS, and JavaScript.
This project allows users to create, read, update, and delete student records efficiently.

Features
➕ Add new student records
📋 View all students in a structured table
✏️ Edit existing student details
❌ Delete student records
🔍 Simple and clean UI for easy navigation
🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL (XAMPP)
Server: Apache (via XAMPP)

⚙️ Installation & Setup
Install XAMPP
Start Apache and MySQL

Place project folder inside:

htdocs/
Create database in phpMyAdmin
Import database.sql
Update database credentials in config.php
Open in browser
